# website:
----------

### Installation

- Clone this project
- Install module dependencies by run this script in terminal
  `npm install`
- Finally run the app.
  `npm run dev`
- Navigate to  [http://localhost:8000](http://localhost:8000)

> env:

```ruby
- node: v16.13.1+
- npm: 8.1.2

```

### Deployment

- First you need to build the production assets first
  `npm run build`
- Then start the app
  `npm start`
- Navigate to  [http://localhost:8000](http://localhost:8000)

> deploy to Github Pages:

```bash
# then check dist/
npm run generate

```
